```
给你两个二进制字符串，返回它们的和（用二进制表示）。

输入为 非空 字符串且只包含数字 1 和 0。

 

示例 1:

输入: a = "11", b = "1"
输出: "100"
示例 2:

输入: a = "1010", b = "1011"
输出: "10101"
 

提示：

每个字符串仅由字符 '0' 或 '1' 组成。
1 <= a.length, b.length <= 10^4
字符串如果不是 "0" ，就都不含前导零。

链接：https://leetcode-cn.com/problems/add-binary

```
class Solution {
public:
    string addBinary(string a, string b) {
        string ret;

        reverse(a.begin(), a.end());
        reverse(b.begin(), b.end());

        int n = max(a.size(), b.size());
        int tmp = 0;

        for (int i = 0; i < n; ++i) {
            if (i < a.size()) {
                tmp += (a[i] == '1' ? 1 : 0);
            }
            if (i < b.size()) {
                tmp += (b[i] == '1' ? 1 : 0);
            }
            
            if (tmp % 2) {
                ret.push_back('1');
            }
            else {
                ret.push_back('0');
            }

            tmp /= 2;
        }

        if (tmp) {
            ret.push_back('1');
        }

        reverse(ret.begin(), ret.end());

        return ret;
    }
};
